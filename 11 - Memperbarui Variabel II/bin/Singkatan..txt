x = x + 10; Menjadi x += 10; 
x = x - 10; Menjadi x -= 10;
x = x * 10; Menjadi x *= 10;
x = x / 10; Menjadi x /= 10;
x = x % 10; Menjadi x %= 10;

Saat Anda menambah atau mengurangi 1, 
Anda dapat mempersingkatnya dengan ++ dan --. 
x++ sama dengan x += 1, dan x-- sama dengan x -= 1. 
Ini adalah cara penulisan yang lebih mudah, jadi mari kita ingat.

x = x + 1; Menjadi x += 1; -> x++; Menambah    1
x = x - 1; Menjadi x -= 1; -> x--; Mengurangi  1
